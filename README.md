# react-redux-crud

## Getting Started

### Launching the Mongo/Express/Node server
```mongod --dbpath /data/db```
```cd react-crud-server```
```yarn install```
```yarn start```
```https://localhost:3001/```

### Launching the client React/Redux server
```cd react-crud-client```
```yarn install```
```yarn start```
```https://localhost:3000/```
